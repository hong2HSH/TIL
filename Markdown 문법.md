# Markdown 문법



## 문단 헤더 (6단까지 가능)

```
# 1단 헤더
## 2단 헤더
```



## 목록

```
* 순서가 없는 목록
- 순서가 없는 목록
1. 순서가 있는 목록
```



## 글자 모양

```
**굵게**
__굵게__
*기울게*
_기울게_
```
